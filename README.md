# Digital Agency & Virtual Institute Landing Page

A modern, high-conversion landing page for a Pakistani Digital Agency + Virtual Institute.

## Features

- **Modern Dark Theme** with premium purple gradient accents
- **Fully Responsive** - Works perfectly on mobile, tablet, and desktop
- **Fast Loading** - Optimized CSS and JavaScript
- **SEO Friendly** - Proper meta tags and semantic HTML
- **Smooth Animations** - Scroll-triggered animations and transitions
- **WhatsApp Integration** - Floating button and multiple CTAs
- **Trust Building** - Stats counter and social proof elements

## Sections

1. **Hero Section** - Bold headline with main CTA
2. **About Section** - Agency introduction with key features
3. **Courses Section** - 5 course cards with icons
4. **Services Section** - Professional agency services
5. **Community CTA** - Main WhatsApp community call-to-action
6. **Trust Section** - Animated statistics
7. **Footer** - Contact info and social links

## Setup Instructions

1. **Update WhatsApp Link**
   - Replace `YOUR_COMMUNITY_LINK` in `index.html` with your actual WhatsApp community invite link
   - Find and replace all 3 instances (floating button, hero CTA, and community section)

2. **Update Contact Information**
   - Edit email, phone, and address in the footer section
   - Update social media links

3. **Customize Content**
   - Modify text content as needed
   - Update course descriptions
   - Change service offerings

4. **Deploy**
   - Upload all files (index.html, style.css, script.js) to your web hosting
   - Or use platforms like Netlify, Vercel, or GitHub Pages for free hosting

## Customization Tips

### Colors
Edit CSS variables in `style.css`:
```css
:root {
    --primary-color: #6C63FF;
    --secondary-color: #25D366;
    --dark-bg: #0F0F1E;
    --dark-card: #1A1A2E;
}
```

### Fonts
Change the Google Fonts import in `index.html` head section.

### Add Analytics
Add your Google Analytics or Facebook Pixel code in the `<head>` section of `index.html`.
